### 项目介绍  
这是一个记录**个人吃喝玩乐日常**的静态网站，包含 4 个页面：  
- 首页：展示近期计划和生活标语  
- 美食记录：分享美食探店体验，支持提交推荐  
- 游玩足迹：记录旅行打卡故事  
- 关于我：介绍个人爱好和小档案  


### 技术实现  
- **HTML5**：使用 `header`/`nav`/`main`/`footer`/`section`/`figure` 等新语义化标签  
- **CSS**：统一多页面风格，实现卡片布局、响应式适配（基础版，可扩展媒体查询）  
- **功能覆盖**：包含表格（近期计划）、表单（美食推荐）、图片展示
